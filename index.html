TimeSpanJS

A time span class written in TypeScript

View project on GitHub

TimeSpanJS is a simple, small time span library written in TypeScript. It supports parsing and formatting time spans.

You can find the documentation under https://hellcracks.github.io/TimeSpanJS/documentation/
